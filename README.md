# Data Science Masters Pre-Course Test Repository

Welcome to the test repository for the Data Science Masters Pre-Course! 🎓

## Purpose

This repository is designed for students enrolled in the pre-course for the Masters degree in Data Science. It serves as a practice environment where you can:

- Learn how to clone a Git repository
- Practice running Python scripts
- Familiarize yourself with basic data science concepts
- Experiment with code modifications

**Important Note:** This repository is for learning and practice only. It is **not designed to be pushed to** or modified on GitHub. Please use it locally for your learning experience.

## Repository Contents

- **`.gitignore`** - Comprehensive gitignore file for Python projects (blocks common file types)
- **`hello_world.py`** - Simple Python script demonstrating basic programming concepts
- **`simple_data_analysis.py`** - Introduction to basic data analysis with Python
- **`README.md`** - This file with instructions and information

## Getting Started

### Prerequisites

Make sure you have the following installed on your system:
- Python 3.6 or higher
- Git

### How to Clone and Use This Repository

1. **Clone the repository** to your local machine:
   ```bash
   git clone https://github.com/DataScienceLabFHSWF/pre_course_test_repo.git
   ```

2. **Navigate to the repository directory**:
   ```bash
   cd pre_course_test_repo
   ```

3. **Run the sample scripts**:
   
   **Hello World Script:**
   ```bash
   python hello_world.py
   ```
   
   **Simple Data Analysis Script:**
   ```bash
   python simple_data_analysis.py
   ```

## What You Can Learn

### From `simple_data_analysis.py`:
